# Task 1: Calculator
A fully functional, interactive calculator built using HTML, CSS, and JavaScript.

## Features
* **Interactive Interface:** Clean UI with a grid system for button alignment.
* **Operations:** Performs basic mathematical functions including addition, subtraction, multiplication, and division.
* **Real-time Display:** A dynamic screen to show user input and results.
* **Key Components:** Developed using eventListeners, operators, and if-else logic.
